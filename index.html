PERSONAL PROJECTS

MyMovieList

MyMovieList is a web applicaiton that I developed using Spring Boot while attending classes at the University of Victoria. MyMovieList allows users to search for any movie and it will return relative information about it such as genre, release date, movie cast, and even ratings. Users can also create accounts to like/dislike movies that they have watched, which is all kept track of on their profile page. Click here to check out the live site (inital load may take a while, try reloading if it fails to load) or the github page to learn more.
